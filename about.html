About this journal

A journal that admits the codebase has ghosts

The Haunted Bicycle is the engineering journal of Skill Weaver, an adaptive assessment project. We write about the architecture decisions we made, the ones we regret, the debugging stories worth retelling, and the research behind adaptive testing — in public, as we go.

Most engineering blogs read like press releases: clean narratives where every decision was obviously correct in hindsight. Ours doesn't work that way, because software doesn't work that way. Every system we've built long enough has developed a personality — parts that groan under load, parts nobody remembers writing, parts that only work if you don't look at them directly. We started calling that personality a haunting, and the name stuck.

A system doesn't get haunted by accident. It gets haunted by everyone who touched it and left something behind. — from the first commit message on this repo

The cast

Three recurring characters carry the stories, each standing in for a role every project needs:

Monster — the developer

Well-meaning, occasionally the source of the haunting. Monster writes the code, reads the stack trace, and owns the consequences either way.

Spider — the algorithm

Patient, precise, and indifferent to your intentions. Spider spins the logic that actually decides what happens, thread by thread.

Bicycle — the software system

The thing you're actually riding. Built to move fast, prone to squeaks nobody can locate, held together by decisions made under deadline.

What we cover

  • Architecture — decisions we made about how Skill Weaver is put together, and the tradeoffs behind them.
  • Debugging — postmortems and stories from chasing down bugs, written while the details are still fresh.
  • Adaptive assessment — the research behind item response theory, adaptive testing, and preserving instructor agency in assessment design.
  • Deployment — the unglamorous, load-bearing work of keeping the thing running.
  • Craft — reflections on how we work, not just what we shipped.
